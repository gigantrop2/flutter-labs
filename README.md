SmartCalc - Умный калькулятор
Краткое описание
SmartCalc — это мобильное приложение-калькулятор с дополнительными полезными функциями: история вычислений и конвертер валют, работающий с актуальными курсами через интернет. Приложение простое, интуитивно понятное и решает повседневные задачи.
Основные функции (разбивка по этапам)
Этап 1: ЛР4 (Верстка)
1.	Базовый калькулятор: Экран с цифровыми кнопками (0-9), основными операторами (+, -, *, /, =), кнопкой сброса (C) и десятичной точкой.
2.	Экран истории: Простой список всех предыдущих вычислений.
3.	Экран конвертера валют: Два поля для выбора валюты и ввода суммы, кнопка для выполнения конвертации.
4.	Навигация: Нижнее или боковое меню для переключения между тремя основными экранами.
Этап 2: ЛР5 (Логика)
1.	Логика вычислений: Корректная обработка нажатий кнопок, выполнение арифметических операций, соблюдение порядка действий (если решите усложнить).
2.	Логика истории: Сохранение в память приложения каждой выполненной операции в формате "5 + 3 = 8" и отображение в списке.
3.	Логика навигации: Реализация перехода между экранами.
Этап 3: ЛР6 (Данные)
1.	Локальное хранение: Сохранение истории вычислений в shared_preferences или локальной БД (SQLite), чтобы история не сбрасывалась после перезапуска приложения.
2.	Работа с API: Получение актуальных курсов валют с публичного API (например, exchangerate-api.com или Frankfurter). Запрос выполняется при открытии экрана конвертера.

             [Главный экран: Калькулятор]
          /                      |                         \

•	Верхняя часть: Дисплей (TextField) для отображения вводимого выражения и результата.
•	Основная часть: Сетка кнопок:
C, +/-, %, /
7, 8, 9, *
4, 5, 6, -
1, 2, 3, +
0, ., =
•	Навигация: Кнопки или нижняя панель для перехода на другие экраны.
2. Экран истории
1.	Заголовок: "История вычислений".
2.	Основная часть: ListView или Column с прокруткой, отображающий последние операции (самая свежая сверху). Каждый элемент — Text("5 + 3 = 8").
3.	Действие: При тапе на запись из истории, выражение можно перенести в калькулятор для повторного использования или редактирования.
3. Экран конвертера валют
1.	Поле "Из": DropdownButton для выбора базовой валюты (USD, EUR, RUB и т.д.) и TextField для ввода суммы.
2.	Поле "В": DropdownButton для выбора целевой валюты и TextField (только для чтения) для отображения результата.
3.	Кнопка: "Конвертировать". При нажатии идет запрос к API и расчет.
4.	Элемент состояния: Индикатор загрузки (CircularProgressIndicator) на время запроса к сети.
Чек-лист необходимого функционала для MVP
1.	Экраны: 3 экрана (Калькулятор, История, Конвертер).
2.	Навигация: Реализована через BottomNavigationBar или Drawer.
3.	Интерактивность: Ввод цифр/операторов, ввод суммы для конвертации.
4.	Список данных: История вычислений отображается в виде списка.
5.	Хранение данных: История сохраняется локально (shared_preferences).
6.	Работа с API: Запрос к API курсов валют с экрана конвертера.
7.	Стилизация: Единая цветовая схема (например, темная тема с акцентным цветом для кнопок операций).
